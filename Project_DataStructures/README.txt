binaryInterpolationSearches.c - Program to search for a value in given array of elements (that follow a selected distribution).
The search algorithms are binary, binary-interpolation and exponential-interpolation searches.

AVLtreeFingerSearch.c - Program to create a AVL tree, insert, delete, access and finger search elements to benchmark its time performance.

Project on "Data Structures" course at Univ. of Patras.
